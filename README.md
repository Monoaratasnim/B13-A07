# 🌿 KeenKeeper

## 📌 Description
KeenKeeper is a modern relationship management web app that helps you stay connected with your friends. It allows you to track interactions, monitor relationship goals, and never lose touch with the people who matter most.

---

## 🚀 Technologies Used
- *Next.js (App Router)*
- *React.js*
- *Tailwind CSS*
- *JavaScript (ES6+)*
- *LocalStorage (for data persistence)*
- *Lucide React Icons*
- *Recharts*

---

## ✨ Key Features

### 1. 👥 Friend Management
- View all friends in a clean card layout  
- Detailed profile page for each friend  
- Track status like on-track, almost due, and overdue

---

### 2. 📊 Timeline Tracking
- Log interactions (Call, Text, Video)  
- Automatically stored in local storage  
- Filter timeline by interaction type  

---

### 3. 📈 Smart Dashboard & Stats
- Overview of total friends and relationship health  
- Track engagement and interactions  
- Visual insights for better connection management
